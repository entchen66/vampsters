# 🦇 Moonlight Peaks – Vampster Tracker

An interactive checklist to find and track all **100 Vampsters** in **Moonlight Peaks**.

👉 **[https://entchen66.github.io/vampsters/](https://entchen66.github.io/vampsters/)**

## Features

- **All 100 Vampsters** organized by their 10 in-game regions
- **Screenshot & placeholder system** – known locations show screenshots (with pinch-to-zoom), unknown ones display a stylized placeholder with the almanac icon
- **Bilingual (DE / EN)** – switch languages via flag toggle
- **Auto-collapse & sparkle celebration** when a region is fully collected
- **Custom animated cursor** – a crystal and bat follow the mouse with physics-based wing flapping
- **Progress saved locally** in the browser (`localStorage`) with automatic migration from older versions

Pure HTML5, CSS3 & Vanilla JS – no frameworks, no build step. Responsive grid layout with touch support.
